I. Гомотопия
* Введем *деформационную ретракцию*, переводящую область X в X (A ∈ X) как набор отображений f в R^2,
задающихся вещественным числом ∈ [0; 1], причем f_0 = id, f_1(X) = A, f_t|_A = id, f(t) - непр.
* Введем *гопотопию* как семейство отображений f_t: X -> Y, F(x, t) = f_t(x) - непр. F: X x I -> Y, t ∈ [0; 1],
* f_0, f_1 : X -> Y *гомотопны*, если ∃ гомотопия, связывающая их (f0 ~~ f1)
* Введем *ретракцию*: f|_A = id, f(X) = A
** Тогда: деформационная ретракция X на A - это гомотопия id на X в ретракцию X на A.

* Задача: смапить тор с выколотой точкой в две окружности, пересекающиеся ровно в одной точке
* Задача: R^n\{0} отобразить на S^n, S^n = {|a|=1 | a ∈ R^n}

* f : X -> Y -- *гомотопическая эквивалентность*, если ∃ g : Y -> X и fg ~~ id и gf ~~  id
    X, Y, для кот. ∃ f - г.э., называаются *гомотопически эквивалентными*
    Частный случай: A ∈ X, f_t : X -> X - диф. ретракция
        i: A -> X - включение
        r: X -> A - ретракция
        i°r ~~ id, r°i = id
* Утв: X, Y - экв. => ∃ Z: Z д.р. в X и в Y. Без док-ва, но хотелось бы с ним.
* Цилиндр отображения X x I. Также оно будет множ-вом Z
* Упр: Композиция гомотопич. экв. X->Y, Y->Z - гом. экв. X->Z
    Доказываем, что гомотоп. эквивалентность - эквивалентность
* Упр: f: X -> Y - гом. экв. f ~~ h. Док-ть, что h - гом. экв.
    Лемма: f ~~ h => f°g ~~ (h°g ~~ id) И g°f ~~ (g°h ~~ id) (g из опр. гом. экв.)

II. Клеточные комплексы
* Пр-во X стягивается в точку, если оно гом. экв. {a}
* Какое-то дз про тор, превращающийся в бутылку Клейна

* e^0 - точка, e^1 - отрезок, e^2 - прямоугольник
* Рецепт 2D тора: e^0, 2 x e^1
* Рецепт 3D тора: e^0, 2 x e^1, e^2
* Рецепт 3D сферы: e^0, e^2
* *Клеточный комплекс* - E^(n-1) = {e^k}, k < n
                         E^n = E^(n-1) U {e^n} / ƒ_k, ƒ_k - отобр. склейки : Boundary(S^(n-1)) -> E^(n-1)
* Если E = E^n, n ∈ N, то E - *конечномерный кл. комп.*
* E^1 - граф
* Другой пример - проективная плоскость PR
PR^n - склеивание ячейки e^n и PR^(n-1)

* *Произведение клеточных комплексов* Ea, Eb = {e_a^k x e_b^l}
 Пример: S^1 x S^1 = {e^1, e^0} x {e^1, e^0} = {Точка, Восьмерка, Восьмерка, Тор}

* Факторпространство
(X, A) - CW - пара
X / A - все клетки из X\A U {A_0} - образ для всех клетов из A
ƒ(e^n) = {e^n [e^n ∈ X\A], A_0 [otherwise]}

* Операция Join
AxBxI. Факторизуем все точки вида (a, _, 0) - |A| факторов
                                  (_, b, 1) - |B| факторов
